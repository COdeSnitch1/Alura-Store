# Alura Store - Análisis de Datos

## Descripción
Proyecto de análisis de ventas de las tiendas Alura Store.

## Análisis realizados
- Facturación por tienda
- Ventas por categoría
- Productos más vendidos
- Calificación promedio
- Costos de envío

## Herramientas usadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Cómo ejecutar

1. Descargar el archivo `.ipynb`
2. Abrir en Google Colab
3. Ejecutar las celdas

## Conclusión

El análisis permite entender el rendimiento de cada tienda y observar tendencias en las ventas y en la satisfacción de los clientes.
