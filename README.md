 Alura Store Latam - Análisis de Ventas - Nicolas Galaz

📌 Propósito del Proyecto
Este proyecto tiene como objetivo analizar los datos de ventas de las tiendas de Alura Store Latam.  
Se realiza un estudio completo de:

- Facturación por tienda
- Productos más vendidos
- Calificación promedio de los clientes
- Costo promedio de envíos

El análisis permite identificar insights importantes para la toma de decisiones comerciales.

 🧰 Librerías Utilizadas

- matplotlib: para la visualización de datos en gráficos.

---

 📊 Análisis Realizado

 1. Facturación por Tienda
Se calcula la facturación total por cada tienda y se visualiza en un gráfico de barras:

- Permite comparar rápidamente qué tienda generó más ingresos.

 2. Productos Más Vendidos
Se identifican los **top productos** por cantidad vendida en cada tienda:

- Ayuda a la planificación de inventario y estrategias de marketing.

 3. Calificación Promedio
Se calcula el rating promedio de los clientes por tienda:

- Indica la satisfacción general del cliente con la tienda.

 4. Costo Promedio de Envío
Se obtiene el costo promedio de envío por tienda:

- Permite evaluar la competitividad y rentabilidad de cada tienda.

 5. Comparativo de Facturación vs. Costo de Envío
Gráfico combinado para visualizar relación entre facturación y costo promedio de envío:

- Facilita identificar oportunidades de optimización logística.
